## 1. 如何运行

> node版本 `[v10.15.3]`

> Vue版本 `[3.6.2]`

> yarn版本 `[1.15.2]`

### 1.1 开发环境配置

基于vue cli3脚手架搭建

### 1.2 开发过程

#### 1.2.1 命令

```sh
# 开发
yarn start

# 发布
sudo yarn run build
```
